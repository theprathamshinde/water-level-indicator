This file contain stimulator of water level indicator using Atmega32 (AVR microcontroller)

There are 2 folder wli1 and wli2 (you can choose any folder according to your proteus version)

Folder consist of:
Image 		: Image of circuit digram of water level indicator.
MicroProject    : Its is a proteus file which contain stumilator of water level indicator.
.hex	        : hex file is provided to upload in Atmega32 stimulator.
code		: c program code for water level indicator.


NOTE : wli1 works on Proteus 8.13 Professional
     : wli2 works on Proteus 8/8.13 Professional
